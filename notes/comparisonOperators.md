# COMPARISON OPERATORS

# 'Loose' Comparison Operators

comparison operators return boolean values: true or false

* == 
    * 25 == '25' => true
    * 19 == 19 => true
    * 7 == 9 => false
* !=
    * 25 != '25' => false
    * 19 != 19 => false
    * 7 != 9 => true
    


# 'Strict' Comparison Operators

* ===
    * 25 === 25 => true
    * 25 === '25' => false
    * 6 === 12 => false
* !==
    * 25 !== 25 => false
    * 25 !== '25' => true
    * 6 !== 12 => true
